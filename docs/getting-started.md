# Getting Started

Welcome to Fakt documentation! This section helps you get started with Fakt, the Kotlin compiler plugin for type-safe fake generation.

## What is Fakt?

Fakt is a compile-time fake generation tool that creates type-safe test doubles from your `@Fake` annotated interfaces. It follows Metro-inspired architecture patterns for professional-quality code generation.
